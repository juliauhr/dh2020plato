Explore Plato's Cave in Virtual Reality

A Lightning Talk for DH2020
By Julia Uhr


Escape Plato's Cave is a virtual reality game where the player solves puzzles to open new areas of the cave and eventually escape. A robot follows the player around, reciting passages from Plato's Republic. The aim is to engage students and the public in learning about philosophy in an immersive and interactive environment.

The puzzles are based on geometrical constructions.

The 3D models were made with Oculus Medium, and the game was made using A-Frame, an open-source framework for web VR.

This is a work in progress, but the code will be available on GitHub when it is finished.




Julia Uhr is a Ph.D. student in creative technology and design at the ATLAS Institute, University of Colorado Boulder.
